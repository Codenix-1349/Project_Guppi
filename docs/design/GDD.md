# Project Guppi -- Game Design Document

## Core Concept

Turn-based deep space strategy focused on progression, sector control,
and faction interaction.

------------------------------------------------------------------------

## Gameplay Pillars

1.  Strategic Decision Weight
2.  Meaningful Progression
3.  Sector-Based Expansion
4.  Tactical but Accessible Combat

------------------------------------------------------------------------

## Core Loop

-   Explore
-   Extract resources
-   Upgrade
-   Engage threats
-   Expand influence

------------------------------------------------------------------------

## Progression

-   Level-based ship evolution
-   Visual module upgrades
-   Unlockable units and technologies

------------------------------------------------------------------------

## Combat System

-   Initiative-based resolution
-   Damage types (kinetic, energy, EMP)
-   Armor & shields
-   Tactical targeting options

------------------------------------------------------------------------

## Economy

-   Iron (Base)
-   Titanium (Structural)
-   Uranium (Advanced)
-   Data (Research)

------------------------------------------------------------------------

## Factions

-   Militaristic Dominion
-   Trade Consortium
-   Technocratic Union
-   Nomad Clans

Each with traits, diplomacy behavior, and unique units.

------------------------------------------------------------------------

## Victory Conditions (Planned)

-   Sector domination
-   Diplomatic supremacy
-   Technological transcendence

------------------------------------------------------------------------

## Long-Term Expansion

-   Multiplayer concept (optional)
-   Mod support
-   Procedural galaxy generation
